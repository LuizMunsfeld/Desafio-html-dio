O Que é Html?

(Voltar)

HTML, que significa "Hypertext Markup Language" (Linguagem de Marcação de Hipertexto, em português), é uma linguagem de marcação usada para criar a estrutura básica de páginas web.

De forma resumida, o HTML é responsável por organizar o conteúdo de uma página, indicando como elementos, texto, imagens e links devem ser exibidos no navegador.

A estrutura básica de um código HTML são compostas por:

  1. DOCTYPE html: Indica a versão do HTML que está sendo utilizada, ajudando o navegador a interpretar corretamente o código.
  2. Html: Define o início e o fim do documento HTML.
  3. Head: Contém informações sobre o documento, como o título que aparece na barra de título do navegador e referências a folhas de estilo CSS, scripts JavaScript, entre outros.
  4. Body: Contém o conteúdo principal da página, como texto, imagens, links e outros elementos visíveis no navegador.

CLIENT X SERVER

(Voltar)

Lado do cliente e lado do servidor são termos de desenvolvimento da web que descrevem onde o código do aplicativo é executado. Os desenvolvedores da web também se referem a essa distinção como front-end X back-end

Client

Solicita serviços ou recursos a um servidor, pode ser um programa ou dispositivo que interage com o usuário e envia solicitações ao servidor, alguns exemplos são navegadores da web, clientes de e-mail, entre outros.

Server

Fornece serviços ou recursos ao cliente, aguarda solicitações de clientes, processa-as e fornece os serviços ou recursos solicitados,tambem pode executar tarefas como servir páginas web, gerenciar bancos de dados, processar e-mails, etc.

Tags e Atributos

(Voltar)

As tags são usadas para informar ao navegador a estrutura do site. Ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página.

A principal característica das tags é estarem sempre dentro dos sinais de chevron (sinal de “maior que” e “menor que”), ou seja: < >.

As tags HTML são divididas em dois tipos: as que precisam de fechamento e as que não precisam de fechamento. As tags que precisam de fechamento possuem a sintaxe , já as que não precisam de fechamento possuem como estrutura .

Além disso, uma mesma tag pode receber um ou mais atributos, que possuirá um valor que modifica sua estrutura ou funcionalidade.

Atributos

(Voltar)

Os atributos são usados para personalizar as tags, modificando sua estrutura ou funcionalidade. Igualmente, os atributos são utilizados para atribuir uma classe ou id a um elemento.

A maioria das tags tem seus próprios atributos. Contudo, existem alguns atributos genéricos que podem ser utilizados na maioria das tags HTML.

Alguns atributos basicos são Class, id, style. Entre outros.

Lista ordenadas e não ordenadas

(Voltar)

A principal diferença entre a lista ordenada e a lista não-ordenada fica na maneira com que os itens de cada lista são apresentados a lista ordenada são numeradas, a lista não-ordenada coloca marcadores antes de cada elemento.

Para criar uma lista não ordenada deve ser utilizada a tag (ul) junto com a tag (li), já para usar uma lista Ordenada se usa o elemento (ol) junto com a tag (li).

Links

(Voltar)

Um link HTML é criado usando a tag (âncora), e é fundamental para a navegação em páginas web. O atributo principal é href, que especifica o URL de destino para o qual o link aponta. O texto entre as tags e é o texto do link visível para o usuário. Além disso, o atributo target pode ser usado para determinar como o link deve ser aberto (por exemplo, em uma nova janela ou guia). Links são essenciais para conectar páginas web, permitindo que os usuários naveguem entre diferentes recursos online.
